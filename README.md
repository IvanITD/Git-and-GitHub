# Git-and-GitHub

Welcome to the **Git-and-GitHub** repository! This is a collaborative project created as part of the **Git and GitHub Lab** at SoftUni. The goal of this lab was to practice essential Git workflows — from cloning and committing to handling merge conflicts and collaborating as a team.
